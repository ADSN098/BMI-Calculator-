# BMI Calculator

A simple, responsive Body Mass Index (BMI) calculator built with HTML5, CSS3, Bootstrap 5, and vanilla JavaScript. Enter your height and weight to instantly calculate your BMI.

## ✨ Features

- Clean, responsive UI built with Bootstrap 5
- Real-time BMI calculation based on user input
- Input validation to handle empty or invalid fields
- Mobile-friendly design

## 🛠️ Tech Stack

- **HTML5** – Structure and markup
- **CSS3** – Custom styling
- **Bootstrap 5** – Responsive layout and UI components
- **JavaScript (Vanilla)** – Client-side logic for BMI calculation

## 🚀 Getting Started

1. Clone the repository
```bash
   git clone https://github.com/ADSN098/BMI-Calculator-.git
```
2. Open `index.html` in your browser, or run a local server:
```bash
   py -m http.server 8000
```
   Then visit `http://localhost:8000`
